Welcome to SPA-Electron

This is I believe an easy framework For Electron to build either SPA pages or you can develope in modular mode, I have developed it as a hobby project in my spare time.

This framework allow you to split and organize your HTMLs and codes for easy developement and organization

Some of the features are:

  1. MVC like structure inspired from codeigniter
  2. You can build your app as an SPA (Single Page App) or Modular (VCs "Views & Controllers")
  3. Easy routing between the views
  4. Using ES6 for custom element definition with automatic imports for css and JS
